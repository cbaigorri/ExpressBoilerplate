# ExpressBoilerplate

A starting point for Express based apps using Vagrant, Browserify, Coffeescript, Less and Jade


## Testing

Starting the server:

Dev mode: nf start -j Procfile.dev
Prod mode: nf start

## Testing

Framework is set up with Mocha and Supertest.

npm test or make test
